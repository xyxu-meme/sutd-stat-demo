Background: checking normality with a Q–Q plot

A Q–Q plot lines up each sample value against the normal quantile it “should” match if the data were normal. For the sorted sample x1 ≤ … ≤ xn, the i-th value is paired with probability level pi = i/(n+1):

x-coordinate = Φ−1(pi)  —  theoretical normal quantile
y-coordinate = xi  —  i-th smallest sample value

Straight line ⇒ plausibly normal. Curved (e.g. S-shaped) ⇒ not normal. Click a point below to see how its coordinates are built, one step at a time.

Where both coordinates come from

Both panels share the same probability level pi = i/(n+1) — the sage horizontal line. On the left, that level meets the sample’s own empirical cdf and drops to the sample y-coordinate. On the right, the same level meets the standard normal cdf Φ(z) and drops to the theoretical x-coordinate.

Each point plots (theoretical normal quantile, sample quantile). A straight line indicates the sample plausibly came from a normal distribution. Switch the sample dropdown to compare a normal sample (straight) against exponential or uniform samples (curved).

X-axis: theoretical quantile of the standard normal, Φ−1(i/(n+1)). Y-axis: sample quantile — the i-th smallest value in the sorted sample, xi. The dashed grey line is a visual guide through the data’s own range, not a fitted regression.
