What Does a 95% Confidence Level Really Mean?

Repeat the same confidence-interval procedure and see that about 95% of the resulting intervals capture the fixed true mean μ.

Try it:Choose a population, confidence level, and sample size. Draw many repeated samples. Each sample produces a different confidence interval, while the true mean μ stays fixed. A 95% confidence level describes the long-run proportion of intervals that capture μ—not the probability that one already-computed interval contains μ.

The underlying simulated observations are held fixed. When n changes, each sample mean is recomputed using the first n observations, so both the centre and width of each CI may change. Larger n narrows the CI and concentrates sample means around μ; these effects balance so long-run coverage stays close to the selected level. For non-normal populations, small-sample coverage may differ, but improves as n grows because of the CLT.

What is sample size n?

n is the number of observations in each sample. Each sample produces one sample mean x̄ and one confidence interval. Larger n usually gives a less variable sample mean and a narrower confidence interval.

What does “Draw 100 samples” mean?

It generates 100 separate samples. Each sample contains n observations and produces one confidence interval. For example, if n = 30, Draw 100 samples produces 100 confidence intervals, each based on 30 observations.

How the summary is counted

  • Each interval comes from a different stored random sequence.
  • Contain μ counts intervals covering the fixed true mean; Miss μ counts those that do not.
  • Observed coverage = intervals containing μ ÷ total intervals.
  • Finite-repetition coverage may fluctuate, especially early.
  • Changing n recomputes both sample means and intervals. For a correctly calibrated known-σ procedure, long-run coverage should remain close to the selected level rather than systematically decreasing as n grows.
